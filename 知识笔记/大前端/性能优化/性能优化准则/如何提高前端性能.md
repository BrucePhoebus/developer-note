# 如何提高前端性能？

## 影响性能的因素有哪些？

#### 白屏

1. HTML 和 CSS 的加载及解析速度

2. <head> 内的脚本加载及执行

#### 首屏

1. 图片加载

2. <body> 内的脚本加载及执行

3. render tree 的构建

HTML 的复杂度
CSS 的复杂度
render tree 的绘制（栅格化）

颜色的复杂度
形状的复杂度

##  围绕哪几方面提高？

1. 缩短白屏时间；

2. 加快首屏显示；

3. 尽快监听主要操作的事件。

> 貌似理想指标是：`60 f/s`
