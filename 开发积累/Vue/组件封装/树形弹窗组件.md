# 树形弹框组件

	这个组件是基于element-ui的树形控件和弹框控件进行二次封装使用
	组件功能为弹框显示树形控件，用于选择组织成员(叶子节点)，当然也可以是其他的

## 实现

#### 使用

*引入依赖*

	因为这里使用了element-ui，所以使用前要引入这个依赖

```bash
cnpm install -S element-ui
```

```js
import ElementUI from 'element-ui';
import  Vue from 'vue'
Vue.use(ElementUI)
```

> 注：一般我们都是在main.js入口文件中引入，属于全局引入，当然也可以按需引入

*父组件中使用*

```template
<create-tree-dialog :isShowDialog="isShowTreeDialog" :dialogTitle="'收件人列表'" :dialogWidth="'30%'" :checkedNodeIdList="selectedList.selectedListId" :treeData="treeData" @confirmCallback="confirmCallback" @cancelCallback="cancelCallback"></create-tree-dialog>
```

> 注：这里传的参数比较多，isShowDialog是控制弹框的隐藏和显示；dialogTitle是弹框的标题；dialogWidth是弹框的宽度；checkedNodeIdList是已选中的ID，是树数据结构中的id，类型为数组；treeData传入树数据，类型为数组；确认按钮回调方法confirmCallback；取消、关闭弹框的回调方法cancelCallback

```js
import createTreeDialog from '../../util/createTreeDialog/createTreeDialog'

components: {
	createTreeDialog
}
```

#### 组件代码

```vue
<template>
	<!--树形结构弹框组件-->
	<div class="container">
		<el-dialog
			:modal="false"
			:title="dialogTitle"
			:width="dialogWidth"
			:append-to-body="isAppendToBody"
			:visible.sync="isShowTreeDialog"
			@close="commonTreeCancel">
			<el-tree
				show-checkbox
				highlight-current
				node-key="id"
				:data="treeData"
				ref="cTreeList5w4ey"
				:props="defaultProps"
				:default-expanded-keys="checkedNodeIdList"
				style="height: 100%">
			</el-tree>
			<span slot="footer" class="dialog-footer">
				<el-button size="mini" type="primary" @click="commonTreeClick">确定</el-button>
				<el-button size="mini" @click="commonTreeCancel">取消</el-button>
			</span>
		</el-dialog>
	</div>
</template>
<script>

	export default {
		name: 'create-tree-dialog',
		props:{
			treeData: {
				type: Array,
				default: () => []
			},
			dialogTitle: {
				type: String,
				default: ''
			},
			checkedNodeIdList: {
				type: Array,
				default: () => []
			},
			isShowDialog: {
				type: Boolean,
				default: false
			},
			dialogWidth: {
				type: String,
				default: '30%'
			},
			isAppendToBody: {
				type: Boolean,
				default: false
			}
		},
		data() {
			return {
				isShowTreeDialog: false,
				defaultProps: {
					children: 'children',
					label: 'text',
					isLeaf: 'leaf',
				},
			}
		},
		watch: {
			isShowDialog(newVal, oldVal) {
				this.isShowTreeDialog = newVal;
			}
		},
		methods: {
			commonTreeClick() {
				this.$emit('confirmCallback', this.$refs.cTreeList5w4ey.getCheckedNodes());
			},
			commonTreeCancel() {
				this.$emit('cancelCallback');
			},
		},
		mounted(){
		}
	}
</script>

<style lang="less" type="text/less" scoped>
	.container {
		width: 100%;
		height: 100%;
		overflow: hidden;
	}

	.container /deep/ .el-dialog {
		position: fixed;
		top: 50%;
		left: 50%;
		margin-top: 0 !important;
		transform: translateX(-50%) translateY(-50%);
	}

	.container /deep/ .el-dialog__body {
		height: 500px;
		overflow: auto;
	}
</style>
```

> 注：这种实现方式可能不那个通用，而且只是将element-ui的实现封装了一层而已，基本的操作还是element的操作，当然是可用进行拓展的

#### 优化

	其实element-ui提供的还有很多的功能操作，可以根据需求场景进行添加或去除

*添加功能*

1. 可以实现搜索功能

	添加搜索功能，因为是组织结构树，所以如果数据量大的时候运行用户进行局部搜索无疑是很有用的功能
	这个在element-ui树形控件中也是比较好实现的，因为官方有提供filter方法实现，还有案例

*去除功能*

	最好是确定好使用的场景类型

1. 去除设置宽

	因为获取组织结构这种树形弹框宽度一般都很固定的，数据量很大的时候也不需要多宽，高更是需要定死，内部滑动即可

*优化后的实现*

```vue
<template>
	<!--树形结构弹窗组件-->
	<div class="container">
		<el-dialog
			:modal="false"
			:title="dialogTitle"
			width="30%"
			:append-to-body="isAppendToBody"
			:visible.sync="isShowTreeDialog"
			@close="commonTreeCancel">
			<el-input placeholder="输入关键字" v-model="filterText" class="new-input">
			</el-input>
			<el-tree
				show-checkbox
				highlight-current
				node-key="id"
				:data="treeData"
				ref="cTreeList5w4ey"
				:props="defaultProps"
				:filter-node-method="filterNode"
				:default-expanded-keys="checkedNodeIdList"
				style="height: 100%">
			</el-tree>
			<span slot="footer" class="dialog-footer">
				<el-button size="mini" type="primary" @click="commonTreeClick">确定</el-button>
				<el-button size="mini" @click="commonTreeCancel">取消</el-button>
			</span>
		</el-dialog>
	</div>
</template>
<script>

	export default {
		name: 'create-tree-dialog',
		props:{
			treeData: {
				type: Array,
				default: () => []
			},
			dialogTitle: {
				type: String,
				default: ''
			},
			checkedNodeIdList: {
				type: Array,
				default: () => []
			},
			isShowDialog: {
				type: Boolean,
				default: false
			},
			isAppendToBody: {
				type: Boolean,
				default: false
			}
		},
		data() {
			return {
				isShowTreeDialog: false,

				filterText: '',	// 要查询的关键字
				defaultProps: {
					children: 'children',
					label: 'text',
					isLeaf: 'leaf',
				},
			}
		},
		watch: {
			isShowDialog(newVal) {
				this.isShowTreeDialog = newVal;
				setTimeout(() => {
					this.filterText = '';
				}, 500);
			},
			filterText(newVal) {
				this.$refs.cTreeList5w4ey.filter(newVal);
			}
		},
		methods: {
			// 关键词查询(过滤)方法
			filterNode(value, data) {
				if (!value) {
					return true;
				}
				return data.text.indexOf(value) !== -1;
			},
			// 确认按钮事件
			commonTreeClick() {
				// 确认回调
				this.$emit('confirmCallback', this.$refs.cTreeList5w4ey.getCheckedNodes());
			},
			// 关闭弹框事件
			commonTreeCancel() {
				// 关闭回调
				this.$emit('cancelCallback');
			},
		},
		mounted(){
		}
	}
</script>

<style lang="less" type="text/less" scoped>
	.container {
		width: 100%;
		height: 100%;
		overflow: hidden;
	}

	.container /deep/ .el-dialog {
		position: fixed;
		top: 50%;
		left: 50%;
		margin-top: 0 !important;
		transform: translateX(-50%) translateY(-50%);
	}

	.container /deep/ .el-dialog__body {
		height: 500px;
		overflow: auto;
	}

	.new-input {
		height: 30px;
		margin-bottom: 10px;
	}
</style>
```
